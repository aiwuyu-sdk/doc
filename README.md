### 三、 联合登录配置申请
#### 邮件申请方式
* 将生成好的RSA公钥或者MD5密钥发送至 caojiayao001@aiwuyu.com
* 邮件发送模版
```
  商户名称：xx
  签名方式：RSA\MD5
  签名密钥：XXXXXXX
```
* 配置完成后会收到回执邮件，内容包含“渠道编码”信息
